## 问题
如何在Go语言中定义、初始化及使用结构体类型？

## 解决
1. **定义结构体**：使用 `type` 关键字配合 `struct` 来声明一个新的结构体类型，内部包含一系列字段（Field）。
2. **实例化结构体**：可以通过多种方式创建结构体实例，包括按顺序赋值、字段名赋值或使用 `new` 函数。
3. **访问与修改字段**：使用点号 `.` 操作符来访问或修改结构体实例中的字段值。
4. **结构体指针**：可以直接使用结构体指针访问字段，Go会自动解引用。

## 代码
```go
package main

import "fmt"

// 1. 定义结构体
type Person struct {
	Name string
	Age  int
}

func main() {
	// 2. 实例化方式一：按顺序初始化
	p1 := Person{"Alice", 20}
	fmt.Println(p1)

	// 2. 实例化方式二：指定字段名初始化（推荐）
	p2 := Person{Name: "Bob", Age: 25}
	fmt.Println(p2)

	// 2. 实例化方式三：使用 new 关键字，返回指针
	p3 := new(Person)
	p3.Name = "Charlie"
	p3.Age = 30
	fmt.Println(*p3)

	// 3. 访问和修改字段
	p2.Age = 26
	fmt.Printf("%s is now %d years old.\n", p2.Name, p2.Age)

	// 4. 结构体指针的使用
	var p4 *Person = &Person{"David", 40}
	// Go允许直接使用指针访问字段，无需 (*p4).Name
	fmt.Println(p4.Name)
}
```

## 总结
1. 结构体是Go中聚合不同类型数据的核心方式，类似于其他语言中的类，但不支持继承。
2. 初始化结构体时，使用“键值对”方式（`Field: Value`）可提高代码可读性且不依赖字段顺序。
3. 结构体可以作为值传递，也可以作为指针传递；使用指针传递可以避免大结构体的拷贝开销，提高性能。
